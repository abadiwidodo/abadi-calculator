Calculator

Introduction
----------------

This is calculator module

Setup
----------------
Install from npm
```
npm i abadi-calculator
```
in your js file,
```bash
var calculator = require("abadi-calculator");
```


Methods
------------------

`add(num1,num2)`: num1 + num2

`substract(num1,num2)`: num1 - num2

`multiply(num1,num2)`: num1 * num2

`divide(num1,num2)`: num1 / num2
